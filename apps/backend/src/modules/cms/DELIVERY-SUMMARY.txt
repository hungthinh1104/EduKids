## Data Models: 4 Models ✅

### Topic Model
- Fields: id, name, description, learningLevel, imageUrl, status, tags, createdBy, publishedBy, publishedAt, timestamps
- Indexes: status, createdBy
- Relations: Flashcards, QuizStructures, User (creator/publisher)

### Flashcard Model (Vocabulary)
- Fields: id, topicId, word, definition, example, imageUrl, audioUrl, status, createdBy, publishedBy, publishedAt, timestamps
- Indexes: topicId, status
- Relations: Topic, User (creator/publisher)

### QuizStructure Model
- Fields: id, topicId, title, description, questionText, options (JSON), difficultyLevel, status, createdBy, publishedBy, publishedAt, timestamps
- Indexes: topicId, status
- Relations: Topic, User (creator/publisher)

### AuditLog Model
- Fields: id, adminId, action, entityType, entityId, changes (JSON), timestamp
- Indexes: adminId, entityType+entityId, timestamp
- Relations: User (admin)
